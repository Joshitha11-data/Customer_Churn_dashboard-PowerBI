# Customer_Churn_dashboard-PowerBI

Built an interactive Power BI dashboard to analyze customer churn, track KPIs, and identify key factors affecting retention using telecom data.

📌 Project Overview

Developed an interactive Power BI dashboard to analyze customer churn patterns and track key performance indicators (KPIs) for business decision-making.

🎯 Objective

To monitor churn rate, identify high-risk customer segments, and support strategies for improving customer retention.

📊 Key KPIs

- Total Customers: 7043
- Churn Count: 1869
- Churn Rate: 26.5%

📈 Dashboard Features

- Churn Rate by Contract Type
- Customer Distribution by Services
- Monthly Charges vs Churn
- Interactive Filters for deep analysis

🔍 Key Insights

- Month-to-month contract customers have highest churn
- Higher monthly charges lead to increased churn
- Customers without tech support show higher churn
- Long-term contracts reduce churn

🛠 Tools Used

- Power BI
- DAX (for KPI calculations)
- Data Modeling

📁 Files

- churn-dashboard.pdf
  
📈 Business Impact

Helps businesses track churn in real-time and make data-driven decisions to improve customer retention and revenue.
